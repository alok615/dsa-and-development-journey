- Problem 1: Linear traversal, track max element with a single variable
- Problem 2: Linear traversal, track both max and second max in one pass
- Problem 3: Two pointer approach - one pointer for position, one for iteration

**Complexities:**
- Problem 1: Time - O(n) | Space - O(1)
- Problem 2: Time - O(n) | Space - O(1)
- Problem 3: Time - O(n) | Space - O(1)
