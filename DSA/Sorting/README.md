## Key Concepts

- Bubble Sort - O(n^2)
- Selection Sort - O(n^2)
- Insertion Sort - O(n^2)
- Merge Sort - O(n log n) - Stable
- Quick Sort - O(n log n) average
- Counting Sort - O(n+k)
- Heap Sort - O(n log n)
