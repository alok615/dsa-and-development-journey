## Key Concepts

- Linear traversal for finding max/min
- Two pointer technique for in-place operations
- Prefix sums for range queries
- Kadane's algorithm for maximum subarray
- Dutch National Flag algorithm for sorting 0s, 1s, 2s

## Patterns to Remember

- When asked for max/min: single pass with a variable
- When asked to rearrange in-place: two pointers
- When asked about subarrays: sliding window or prefix sum
- When asked about pairs: sort + two pointers or hash map
