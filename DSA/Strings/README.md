## Key Concepts

- String traversal and manipulation
- Two pointer for palindrome check
- Sliding window for substring problems
- Hashing for anagram detection
- KMP algorithm for pattern matching

## Patterns to Remember

- Palindrome check: two pointers from both ends
- Anagram: sort both strings or use frequency map
- Longest substring without repeat: sliding window + hash set
