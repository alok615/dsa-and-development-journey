## What Is It?

HTML buttons are interactive UI elements used to trigger actions. They can be standalone or inside forms. Three main types: button, submit, and reset.

## Why Is It Important?

Buttons are the primary way users interact with web pages - submitting forms, triggering JavaScript functions, and controlling UI state.

## Key Points

- type="button" - Generic button, used with JavaScript
- type="submit" - Submits the parent form
- type="reset" - Resets all form fields to default
- disabled attribute prevents interaction
- aria-label improves accessibility
- :hover and :active pseudo-classes add interactive feedback
